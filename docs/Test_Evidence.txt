Validation-->

Student ID Validation:
Before performing search, update, or delete operations, the system verifies whether the entered student ID exists.

Empty List Validation:
If no students are stored, the system displays an appropriate message.

Age Validation:
Age must be greater than zero; invalid age values are not accepted.

Menu Choice Validation:
Invalid menu choices display an error message and prompt the user again.
